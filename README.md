rfriends_termuxはandroidスマートホン上にrfriends3をインストールするスクリプトです。  

cd ~/  
termux-setup-storage  
sudo apt install git  
git clone https://github.com/rfriends/rfriends_termux.git  
cd termux 
sh rfriends3_termux.sh
